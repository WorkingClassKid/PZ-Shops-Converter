# PZ-Shops-Converter
Python script to update the configuration file for the Shops mod from b41 (by Noir's) to b42 (by eScape)

Original Shops bod by Noir (B41): https://steamcommunity.com/sharedfiles/filedetails/?id=2840330802

Updated Shops mod for B42 by eScape: https://steamcommunity.com/sharedfiles/filedetails/?id=3631768689


# Usage

`python3 converter_shops.py input.lua output.lua`


`python3 converter_forsale.py input.lua output.lua`
